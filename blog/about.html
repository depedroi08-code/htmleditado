Sobre este blog

Mi blog de tecnología es un espacio personal donde registro lo que voy aprendiendo sobre desarrollo web, programación y herramientas digitales. La idea es usar este sitio como ejercicio práctico para la materia de HTML y, al mismo tiempo, como un lugar para reflexionar sobre mi propio proceso de aprendizaje.

Los temas van desde conceptos muy básicos, como la estructura de una página en HTML, hasta ideas más avanzadas relacionadas con la publicación de proyectos en la web y el uso de plataformas como GitHub Pages.

Sobre mí

Soy Ignacio, estudiante y entusiasta de la tecnología. Me interesa entender cómo funcionan las cosas por dentro, especialmente en el mundo digital. A través de este blog, quiero practicar la escritura de código y a la vez mejorar mi forma de explicar lo que aprendo.

Además de la programación, me gusta explorar nuevas herramientas, aprender de tutoriales y compartir recursos con otras personas que también están empezando en este camino.

Experiencia y logros recientes

  • Curso de introducción a HTML y CSS completado.
  • Primera página web publicada usando GitHub Pages.
  • Prácticas de programación básica con JavaScript.
  • Participación en proyectos académicos relacionados con tecnología y educación.
  • Creación de este blog como parte de la sección “HTML Project”.
